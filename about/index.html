Sobre Nosotros

EditoresLand

EditoresLand es un conjunto de guías y documentación en español sobre tus editores de texto favoritos su propósito es difundir y agrandar la documentación en la comunidad hispana sobre cada uno de los editores de texto una herramienta principal en el desarrollo del software

Herramientas ?

Nos hemos apoyado de varias herramientas las cuales son:

  • Zola Un generador de sitios estáticos para la página principal
  • MdBook Un generador sitios estáticos usado en el libro principal creado por RustLang
  • Github Pages El hosting para todas las páginas web de EditoresLand
  • MdFormat El formatter y linter para las páginas markdown
  • EditorConfig Para estilizar y mantener un estilo en los documentos usamos EditorConfig para usar solo tabs o spaces
